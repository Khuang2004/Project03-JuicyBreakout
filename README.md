# Project03-JuicyBreakout
2/28/2026

Juicy Breakout is a simple brick breaking game where the player moves a paddle to keep the ball from falling and clear all the bricks on the screen. The gameplay itself is straightforward, just try to destroy all the bricks while not losing the three lives, means that you have three chances.

## Implementation
- Changing the colors
- Animate the elements
- Animate elements when they are hit
- Shake the screen when something good happens
- Shake the screen when an error occurs
- Add sound effects & Add music
- Break the elements when they are hit
- Add a comet trail to the ball
- Add eyesballs
- Make the eyes follow the action
- You may also add shaders.

## Future Development
- Could add more moving elements, like the bricks start moving down, which can increase the diffculty.
- Mutiple hits before the bricks are destroyed, one brick needs more than one hit to be destroyed.
